# 🎯 Ball Hunter PRO+

Um mini game interativo desenvolvido com HTML, CSS e JavaScript puro, focado em reflexo, precisão e tempo de resposta.

---

## 🎮 Funcionalidades

- 🎯 Clique na bolinha para marcar pontos
- ⏱️ Cronômetro de 30 segundos
- ❤️ Sistema de vidas
- ⚡ Dificuldade progressiva (velocidade aumenta com o tempo)
- 🔊 Efeitos sonoros (acerto e erro)
- 🏆 Ranking com armazenamento local (LocalStorage)
- 👤 Nome do jogador
- 🎨 Interface moderna e responsiva

---

## 🖥️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 🧠 Conceitos aplicados

- Manipulação do DOM
- Eventos (click, interação do usuário)
- Estruturação de código (estado do jogo)
- Uso de LocalStorage
- Lógica de jogo (pontuação, tempo, vidas)
- Responsividade básica
- UX/UI (feedback visual e sonoro)

---

🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, incluindo manipulação de elementos, eventos, lógica de programação e armazenamento de dados no navegador.
